Browser::Open
=============

This is a Perl 6 port of Browser::Open http://metacpan.org/module/Browser::Open. 

## Installation

To install it using Panda (a module management tool bundled with Rakudo Star):

    panda update
    panda install Browser::Open

## Testing

To run tests:

    prove -e perl6
